# Gym Apparel Store

A full-stack e-commerce web app for gym/fitness apparel, built as a portfolio project.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- PostgreSQL + Prisma
- Auth.js
- Stripe

## Status
🚧 In progress — see commit history for build stages.

## Features (planned)
- Product listings & detail pages
- Shopping cart
- User authentication
- Checkout with Stripe
